
# Rust语句和表达式的区别
语句不会返回一个结果。
表达式会返回一个结果。**即表达式总要返回值**。


```rust
#![allow(unused)]
fn main() {
    fn add_with_extra(x: i32, y: i32) -> i32 {
        let x = x + 1; // 语句
        let y = y + 5; // 语句
        x + y // 表达式
    }
}
```

**表达式总要返回值** 

**表达式不能包含分号，一旦在表达式后加上分号，它就会变成一条语句** 

**表达式如果不返回任何值，会隐式地返回一个 `()`**
